Hi, there! I'm Brian Storti, and I like to create things with code.

I've worked with small companies, I've worked with huge companies, and I even created my own company some years ago, but now I'm a consultant and developer at ThoughtWorks, where I work with smart people, solving interesting problems, which is pretty cool.

I've experienced a bunch of different technologies already, but the ones I'm more active now are Ruby, Java and JavaScript. I'm also starting to write some really bad Clojure in my spare time, hoping that it will get better soon.

I also believe that the only way to indent code is with spaces, so if you use tabs, please stop.

You can follow me on twitter, read some of my code on github or send me an email. If you want things to get serious, you can also find me on LinkeIn.
